Text Sentiment Analysis
  using CNN-LSTM & LSTM-CNN

  • Tech Stack: Python, Flask, Keras, TensorFlow, HTML, CSS, Bootstrap
  • Github URL: Project Repository
  • Live Demo: Click here to try it out

Developed deep learning architectures combining Convolutional Neural Networks (CNN) and Long Short-Term Memory (LSTM) networks for sentiment classification of text data with high accuracy and robustness.

Built a responsive web interface using Flask and Bootstrap that allows users to input custom text and receive real-time sentiment predictions (positive, negative, neutral) powered by trained CNN-LSTM and LSTM-CNN models.

Conducted extensive evaluation of model performance using accuracy, precision, recall, and F1-score metrics, and visualized results through confusion matrices and performance graphs.
