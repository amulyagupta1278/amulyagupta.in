Age Invariant Face Recognition

  • Tech Stack: Python, OpenCV, Keras, TensorFlow, HTML, CSS, Bootstrap
  • Github URL: Project Repository
  • Live Demo: Click here to try it out

Developed a deep learning-based face recognition system that maintains high recognition accuracy across different age groups and aging variations.

The system uses pre-processing techniques to normalize facial features and enhance age-invariant properties, followed by a CNN architecture trained on age-diverse datasets.

Integrated OpenCV for real-time face detection and recognition, and built a user-friendly web interface for demonstration and testing.
